# loginPacientes-MakiSanTech
Login sencillo desarrollado con React JS para el proyecto de odontología de MakiSan Tech. Puede añadir base de datos y/o cualquier componente para enriquecer el componente. Inicializar proyecto con los comandos npm i, y luego con npm run dev. Luego encenderlo con este último siempre que se lo necesite.
